  • :has
    ある要素の後に続いている場合にスタイルを適用できたりする
    
        .container{
            ul{
                li :has(h1){
                    color: red;
                }
            }
        }
以下はdivの次にpタグがある場合は赤色になる例

pタグ（赤くならない）

divタグ（赤くなる）

pタグ（赤くならない）

divタグ（赤くならない）
divタグ（赤くならない）

div in pタグ（赤くならない）
