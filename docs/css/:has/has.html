  • hasと+セレクタの合せ技
  • 参考
    :hasセレクタは親要素が指定している子要素を持っている場合に親にスタイルを適用できるセレクタ

    なのでflex,gridなどのレイアウトを組む際に便利なセレクタ

    また、:has(要素1, 要素2)と定義することで要素１と要素２のどちらを持っている場合スタイルを適用することができる

    
        .container{
            ul{
                li :has(h1){
                    color: red;
                }
            }
        }
以下はli内にh1がある場合は赤色になる例

  • H1赤くなる

  • H2赤くならない

  • H3赤くならない

  • H4赤くならない
