構文

    

        /* 結合子の前後の空白はオプションですが、推奨されます。 */
        先行する要素 ~ 選択される要素 { スタイルプロパティ }
    

    後続兄弟結合子(~)は、１つ目に定義された要素の後（直後以外にも反応する）に

    ２つ目に定義された要素が来た場合に

    ２つ目に定義された要素にスタイルを適用するセレクタ
    
        .container{
            div ~ p{
                color: red;
            }
        }
以下はdiv要素の後続にpタグがある場合は赤色になる例

pタグ（赤くならない）

divタグ

pタグ(赤くなる)

div in pタグ（赤くならない）

pタグ(赤くなる)
